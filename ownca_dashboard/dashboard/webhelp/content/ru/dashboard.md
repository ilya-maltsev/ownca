# Обзорная панель

Стартовая страница интерфейса (`/`). Показывает актуальную сводку по
инфраструктуре удостоверяющих центров.

## Карточки сводки

В верхней части — карточки с агрегированными показателями:

* **Удостоверяющие центры** — общее число, корневые и подчинённые.
* **Сертификаты** — действующие, отозванные, истёкшие.

Цифры рассчитываются в реальном времени из БД и обновляются при каждом
посещении страницы.

## Таблицы

* **Удостоверяющие центры** — список с числом выданных / активных / отозванных сертификатов.
* **Истекают в ближайшие 30 дней** — список сертификатов с наступающим сроком.
* **Недавно выданные** — последние 10 сертификатов.
