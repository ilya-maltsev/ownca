# Управление паролем

Интерфейса политики паролей нет. Пароль суперпользователя
управляется через переменные окружения.

## Установка пароля

Задайте `DASHBOARD_ADMIN_PASSWORD=<новый_пароль>` в `docker-compose.yml`
(или в среде) и перезапустите контейнер — значение применяется
автоматически при следующем старте.

## Надёжность пароля

Стандартные валидаторы всегда активны:

* Минимальная длина.
* Проверка на наличие в словаре распространённых паролей.
* Запрет паролей из одних цифр.
* Проверка схожести с именем пользователя.
