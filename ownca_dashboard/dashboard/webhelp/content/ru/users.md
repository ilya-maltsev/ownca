# Учётная запись администратора

OwnCA работает с единственной учётной записью суперпользователя.
Интерфейса управления пользователями нет — учётная запись создаётся
автоматически из переменных окружения.

## Учётные данные

| Переменная | По умолчанию | Назначение |
|---|---|---|
| `DASHBOARD_ADMIN_USER` | `admin` | Логин. |
| `DASHBOARD_ADMIN_PASSWORD` | `admin` | Пароль. |

Задайте эти значения в `docker-compose.yml` (или в среде) до первого
запуска, затем сразу измените на надёжные.

## Инициализация

При каждом старте контейнера учётная запись администратора создаётся
или обновляется автоматически:

1. Суперпользователь создаётся, если его не существует.
2. Пароль приводится к текущему значению `DASHBOARD_ADMIN_PASSWORD`.
3. Устаревшие учётные записи без прав суперпользователя удаляются.

## Смена пароля

Обновите `DASHBOARD_ADMIN_PASSWORD` и перезапустите контейнер —
инициализация применит новое значение.

## Безопасность сессий

Сессии защищены настройками куки: `HttpOnly`, `SameSite=Strict`,
`Secure`, кука истекает при закрытии браузера.
