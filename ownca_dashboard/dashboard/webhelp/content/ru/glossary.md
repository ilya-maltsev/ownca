# Глоссарий

| Термин | Определение |
|---|---|
| **AIA** | Authority Information Access — расширение X.509 с указанием URL OCSP-респондера и места загрузки сертификата УЦ. |
| **ASN.1** | Abstract Syntax Notation One — нотация описания структур данных, в которой формализованы криптообъекты X.509. |
| **CA** | Certification Authority — удостоверяющий центр. |
| **CDP** | CRL Distribution Points — расширение X.509 со списком URL, по которым публикуется CRL. |
| **CN** | Common Name — атрибут DN, обычно содержит имя владельца, e-mail или DNS-имя сервиса. |
| **CRL** | Certificate Revocation List — подписанный УЦ список отозванных сертификатов. |
| **CSR** | Certificate Signing Request — запрос на сертификат (PKCS#10). |
| **DER** | Distinguished Encoding Rules — бинарная кодировка ASN.1, единственный валидный способ кодировать сертификаты. |
| **DN** | Distinguished Name — последовательность RDN-атрибутов, идентифицирующих subject или issuer. |
| **EKU** | Extended Key Usage — расширение X.509 со списком OID-ов прикладных назначений ключа. |
| **End-entity** | Конечный сертификат — выданный пользователю или сервису, не УЦ. |
| **Issuer** | Сторона, подписавшая сертификат. |
| **Key Usage (KU)** | Расширение X.509 с битовой маской допустимых операций для ключа. |
| **OCSP** | Online Certificate Status Protocol — онлайн-проверка статуса сертификата. |
| **OID** | Object Identifier — уникальный идентификатор; в X.509 каждое расширение имеет свой OID. |
| **PEM** | Privacy-Enhanced Mail — текстовая обёртка над DER (BASE64 + заголовки `-----BEGIN ...-----`). |
| **PKCS#7** | Криптографический контейнер с одним или несколькими сертификатами. |
| **PKCS#10** | Формат CSR. |
| **PKCS#12** | Контейнер с сертификатом и закрытым ключом, защищённый паролем. |
| **Path validation** | Процесс проверки цепочки сертификатов до доверенного корня. |
| **PKI** | Public Key Infrastructure — совокупность УЦ, политик, ПО и процессов выпуска сертификатов. |
| **Profile** | Профиль — в OwnCA шаблон расширений сертификата. |
| **RDN** | Relative Distinguished Name — один атрибут DN. |
| **Root CA** | Корневой УЦ — самоподписанный сертификат вершины цепочки. |
| **SAN** | Subject Alternative Name — расширение X.509 с альтернативными именами владельца: DNS, IP, e-mail, URI. |
| **Serial number** | Уникальный в рамках УЦ номер сертификата. |
| **SIA** | Subject Information Access — расширение X.509 с URL репозитория сертификатов УЦ-subject. |
| **SKI / AKI** | Subject Key Identifier / Authority Key Identifier — расширения с производным от ключа идентификатором. |
| **Subject** | Сущность, которой выдан сертификат. |
| **TSA** | Time-Stamping Authority. |
| **VKO** | Алгоритм выработки общего ключа в ГОСТ Р 34.10-2012. |
| **X.509** | Стандарт ITU-T для сертификатов открытых ключей. |
| **ГОСТ Р 34.10-2012** | Российский стандарт ЭП на эллиптических кривых. |
| **ГОСТ Р 34.11-2012 («Стрибог»)** | Российская хеш-функция, 256 и 512 бит. |

См. также [Основные понятия PKI](concepts.md), [Структура X.509-сертификата](x509_overview.md).
